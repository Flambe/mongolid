DataMapper

class DataMapper

The DataMapper class will abstract how an Entity is persisted and retrieved from the database.

The DataMapper will always use a Schema trought the SchemaMapper to parse the document in and out of the database.

Properties

string $schemaClass Name of the schema class to be used
Schema $schema Schema object. Will be set after the $schemaClass
