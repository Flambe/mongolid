EmbeddedCursor

class EmbeddedCursor implements Iterator

This class wraps the query execution and the actuall creation of the driver cursor. By doing this we can, call 'sort', 'skip', 'limit' and others after calling 'where'. Because the mongodb library's MongoDB\Cursor is much more limited (in that regard) than the old driver MongoCursor.
