ActiveRecord

class ActiveRecord

The Mongolid\ActiveRecord base class will ensure to enable your entity to have methods to interact with the database. It means that 'save', 'insert', 'update', 'where', 'first' and 'all' are available within every instance.

The Mongolid\Schema that describes the entity will be generated on the go based on the $fields.

Traits

Attributes
This trait adds attribute getter, setters and also a usefull fill method that can be used with $fillable and $guarded properties to make sure that only the correct attributes will be set.
Relations
It is supossed to be used in model classes in general

Properties

string $collection Name of the collection where this kind of Entity is going to be saved or retrieved from
boolean $dynamic The $dynamic property tells if the object will accept additional fields that are not specified in the $fields property. This is usefull if you doesn't have a strict document format or if you want to take full advantage of the "schemaless" nature of MongoDB.
